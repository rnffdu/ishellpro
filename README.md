## 主要功能一览

**SSH 连接管理**
多配置保存，支持密码、密钥、证书、键盘交互等多种认证方式，连接管理界面简洁直观。

**内置 SFTP**
不需要再另开 WinSCP 或 FileZilla。内置 SFTP 文件管理器，树形目录浏览，支持拖拽上传，文件编辑内置代码高亮和行号显示，还支持断点续传——传大文件时网络抖一下不用从头来。

**AI 辅助操作**

**命令市场**
内置命令模板市场，可以搜索下载常用命令模板，也可以分享自己的命令。批量对多台主机执行同一条命令也是支持的。

**全协议覆盖（3.0 版本新增）**
3.0 版本还加入了 RDP/VNC、Telnet、串口支持，终于能把 Windows 远程桌面和 SSH 放在同一个工具里管了。
